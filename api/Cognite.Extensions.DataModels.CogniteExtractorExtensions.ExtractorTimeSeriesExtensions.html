EnsureTimeSeriesExistsAsync<T>(CoreTimeSeriesResource<T>, IEnumerable<SourcedNodeWrite<T>>, int, int, RetryMode, SanitationMode, CancellationToken)

Ensures that all time series in timeSeriesToEnsure exists in CDF. Tries to create the time series and returns when all are created or have been removed due to issues with the request. If any items fail to be created due to missing asset, duplicated externalId, duplicated legacy name, or missing dataSetId, they can be removed before retrying by setting retryMode Timeseries will be returned in the same order as given in timeSeriesToEnsure

GetOrCreateTimeSeriesAsync<T>(CoreTimeSeriesResource<T>, IEnumerable<InstanceIdentifier>, Func<IEnumerable<InstanceIdentifier>, IEnumerable<SourcedNodeWrite<T>>>, int, int, RetryMode, SanitationMode, CancellationToken)

If one or more do not exist, use the buildTimeSeries function to construct the missing time series objects and upload them to CDF using the chunking of items and throttling passed as parameters If any items fail to be created due to missing asset, duplicated externalId, duplicated legacy name, or missing dataSetId, they can be removed before retrying by setting retryMode
