List of accepted config versions, null accepts all
System.String appId

AppId to append to requests to CDF
System.String userAgent

User agent on form Product/Version

True if the extractor uses a state store
System.Boolean addLogger

True to restart extractor if it crashes or terminates, using exponential backoff
CancellationToken token

System.Nullable<ServiceCollection> extServices

Optional pre-configured service collection

Optional logger to use before config has been loaded, to report configuration issues
TConfig config

Optional pre-existing config object, can be used instead of config path.
